# Análise de Concorrência — Ferramentas de Supplier Query

## Concorrentes Diretos

### 1. Linker (Brasil)
- **Preço**: R$ 2.500/mês (até 500 consultas)
- **Força**: Integração nativa com ERPs (SAP, Oracle)
- **Fraqueza**: UI complexa, onboarding lento
- **Market Share**: ~15% no segmento enterprise

### 2. SupplierHub (EUA)
- **Preço**: $99/mês (usuário)
- **Força**: UX excelente, API pública
- **Fraqueza**: Sem dados da Receita Federal, suporte em inglês
- **Obs**: Não atende bem o mercado brasileiro

### 3. Solução Interna (Planilhas + Email)
- **Custo**: Baixo custo financeiro, alto custo de tempo
- **Problemas**: Sem rastreabilidade, erros manuais, sem escala
- **Status quo atual**: 78% dos times ainda usam este método

## Diferenciais Propostos
1. Consulta unificada (CNPJ + CPF + Nome) em única interface
2. Dados enriquecidos automaticamente (Receita + Serasa + Interno)
3. Histórico completo com linha do tempo
4. Alertas proativos de documentação
5. Custo acessível para PMEs

## Posicionamento
"O primeiro sistema brasileiro de consulta inteligente de fornecedores que unifica dados públicos e internos em uma experiência simples e rápida."

## TAM no Brasil
- ~50.000 empresas com supply chain estruturado
- Potencial: 500 clientes no primeiro ano
- Ticket médio estimado: R$ 1.200/mês
